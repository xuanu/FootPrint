# footprint
足迹，地图打点应用
> 简单的想记录一下自己每天的足迹，顺便学习一下Kotlin的语法。

#### 问题
1. GPS会消耗大量电量,所以没有采用GPS定位
2. 每分钟记录一次，会产生大量数据，存入/sdcard/footprint/db/loaction.db数据库
